## 1. Философия. 

«Природа наделила нас двумя глазами, двумя ушами, но лишь одним языком, дабы мы смотрели и слушали больше, чем говорили.» — Сократ.

Когда есть ресурс системы, система функционирует. Ресурс поставляется отдельно от функции, таковы правила экономики — управлять ресурсами. Система предоставляет продукт и на этом точка.

## 2. Краткое описание авторской задумки.

Далее план игрового физического движка. Программа состоит из трёх базовых блоков, именно программный интерфейс приложения, ввод и вывод — это основа метода далее:
1. Схлопывание алгебраической функции. Алгебраическое ядро.
2. Методы двигателя (игровой движок): математические объекты представления функций матрицы (алгебраическое ядро).
3. Ввод: список названий (вещей) с координатами.
4. Вывод: проекция векторного пространства "хлопков".
5. Хлопок - это схлопывание алгебраической функции ядра.

## 3. Метод.

2. Обеспечение технологий поддержания жизнедеятельности. Например диагностирование типа аллергии.
